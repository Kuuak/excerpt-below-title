Move the excerpt metabox below the post title

== Description ==

Simply force display and move the excerpt input box right below the title and before the content.

== Installation ==

1. Upload `excerpt-below-title.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
